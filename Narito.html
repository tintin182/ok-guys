Activity 1

  1. What are HTML tags? (Own definition)

    HTML tags are the building blocks of web pages. They are used to define and structure content on a webpage, such as text, images, links, and other elements. Tags typically come in pairs, with an opening tag and a closing tag.

  2. Give at least 10 examples of HTML tags and explain them one by one, why it is important?
    • <html>: Defines the root of an HTML document. It is essential as it wraps all the content of the webpage.
    • <head>: Contains metadata, such as the document title and links to stylesheets. Important for page setup.
    • <title>: Specifies the title of the webpage, shown on the browser tab. Vital for identifying the page.
    • <body>: Contains the main content of the HTML document, displayed on the webpage.
    • <h1> to <h6>: Define headings, used to structure content hierarchically.
    • <p>: Defines a paragraph. Essential for displaying blocks of text.
    • <a>: Defines hyperlinks, enabling navigation between pages.
    • <img>: Embeds images into the webpage. Enhances visual appeal and communication.
    • <ul> and <ol>: Define unordered and ordered lists, respectively. Useful for organizing content.
    • <table>: Defines tables for displaying structured data.
  3. What is the difference between a tag in HTML and an attribute in HTML?

    Tags are elements used to define and structure content in HTML, such as <p> for a paragraph. Attributes provide additional information about tags, such as id or class, to customize their behavior or style.
